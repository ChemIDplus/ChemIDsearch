You may intentionally return limited results for server-side paging. To request small amounts of results at a time, include the rowStart parameter, starting at 1. The number of results in the response is then determined by the pageSize parameter. If pageSize is not included, then the default of 5 is used. For example, with the default pageSize, the first two pages for "RN starts with 50-" using the abbreviated URLs would be:

  • https://chem.nlm.nih.gov/api/data/rn/sw/50-?data=summary&rowStart=1
  • https://chem.nlm.nih.gov/api/data/rn/sw/50-?data=summary&rowStart=6

Sorting and Paging can also be combined. When sorting by descending citations, the first two pages at 10 records each would be:

  • https://chem.nlm.nih.gov/api/data/rn/sw/50-?data=summary&sortBy1=citations&rowStart=1&pageSize=10
  • https://chem.nlm.nih.gov/api/data/rn/sw/50-?data=summary&sortBy1=citations&rowStart=11&pageSize=10

If an API search matches many substances, the results may be returned in batches, even without the rowStart parameter. If a SubstancesResult includes start and end and the requst did not include rowStart, then the server has only sent a single batch back. Additional batches may then be requested by including the batchStart parameter.

For example, let us examine a search on "RN starts with 50-", requesting all ChemIDplus data for every record. The abbreviated URL for that search is: https://chem.nlm.nih.gov/api/data/rn/sw/50-?data=complete which returns: 

{{ '{' }}
	"total": 80,
	"start": 1,
	"end": 25,
	"results": [
		...
	]
}

The search matched 80 substances. We did not request paging, but the result includes start and end. Therefore, we know the server only returned a batch with the first 25 substances. To get the rest, we would need to make multiple requests, each incrementing the batchStart parameter to be one more than the prior end value:

  • https://chem.nlm.nih.gov/api/data/rn/sw/50-?data=complete&batchStart=26
  • https://chem.nlm.nih.gov/api/data/rn/sw/50-?data=complete&batchStart=51
  • https://chem.nlm.nih.gov/api/data/rn/sw/50-?data=complete&batchStart=76
