The following operators are only available when used with the Structure field.

  1. substructure
  2. similarity
  3. exact
  4. flex
  5. flexplus

Similarity searches have a default similarity percent of 80%, but this can be changed whenever a new search is run. The results for similarity searches include the similarity percentage, and the default sort retrieves the most similar substances first.

Structure searches can have one of a few types of Search Terms.

  1. RN (hyphens are optional)
  2. 10-digit System ID
  3. SMILES (case sensitive)
  4. molfile
  5. chime
  6. HELM
