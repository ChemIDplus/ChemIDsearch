On the Create an Expression page, the (automatic) operator is selected by default.

When you search using this operator, the application will:

  1. First run the search using the equals operator. If it finds a match, it returns that search's result.
  2. If it does not find a match, it will instead return the result of searching using the starts with operator.

Note that multi-expression searches may not contain any expression with the (automatic) operator.
