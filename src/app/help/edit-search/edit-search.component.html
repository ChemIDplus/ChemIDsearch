Once you have run a query, you can return to it by clicking the Create Search button again. You can edit your prior search by:

  • Clicking the Add another button to create additional expressions.
  • Clicking an expression's row's Edit button to modify the expression.
  • Clicking an expression's row's Delete button to remove the expression.
  • Clicking the Clear button to remove all expressions and start over.

For any expressions you decide to edit, if while editing it you decide to cancel your changes, you will return to this page with your current query unchanged.
