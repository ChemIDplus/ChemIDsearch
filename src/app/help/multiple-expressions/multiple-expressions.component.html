After generating a single expression, the application displays the Expression matches ... page. Instead of immediately running the search, another option is to create one or more additional expressions.

You can click on the Add another button to create a second expression. On completing the second expression, the expressions list page title will update to Expressions match ... with new totals. The two expressions are ANDed together, such that all matching substance fulfill both expressions.

Any number of additional expressions can be added to the search and all will be true for any matching results.

Whenever there is more than one expression, you also have the option to mark the "Not" checkbox for some of your expressions. For a valid search, at least one expression must not have "Not" checked.
